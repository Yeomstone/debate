문제 해결

PROBLEM

공공데이터포털 접속 불가

데이터센터 화재로 공공데이터포털 API가 장기간 중단되어 계획했던 데이터 수집이 불가능해졌습니다.

SOLUTION

대체 데이터 소스 확보

  • 금융감독원 DART OpenAPI 활용
  • 사업보고서 일괄 다운로드 로직 구현
  • 필요 데이터 필드 자동 추출
