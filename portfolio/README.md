- `portfolio/portfolio.html` - 13장 슬라이드 포트폴리오

---

## PDF로 변환하는 방법

### 방법 1: 브라우저 인쇄 기능 (추천)

1. `portfolio.html` 파일을 브라우저에서 엽니다
2. `Ctrl + P` (인쇄) 누르기
3. **대상**: "PDF로 저장" 선택
4. **여백**: "없음" 선택
5. **배경 그래픽**: 체크 ✅
6. **저장** 클릭

### 방법 2: PowerPoint로 변환

1. 각 슬라이드를 스크린샷으로 캡처
2. PowerPoint에 이미지로 삽입
3. PPT/PDF로 저장
